# Sprint_4
Автоматизированные UI-тесты для тестирования FAQ, а также позитивных сценариев оформления заказа самоката

## Технологии
- Java 11
- Selenium WebDriver 4.38.0
- WebDriverManager 6.3.2
- JUnit 4.13.2
- Maven 3.9.0
- IntelliJ IDEA (или другой совместимый IDE)

## Настройка среды 
1. Клонируйте репозиторий:
https://github.com/tea1311/Sprint_4.git
2. Импортируйте проект в IntelliJ IDEA.
3. Убедитесь, что установлен JDK 11 и настроены зависимости Maven — они подтянутся автоматически по `pom.xml`.

## Запуск тестов
```bash 
mvn clean test
```
